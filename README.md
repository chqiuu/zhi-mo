# 执墨 - AI 新媒体内容创作平台

> 执笔倚墨，AI成文

## 项目简介

执墨是一款 AI 驱动的新媒体内容创作工具，专注于 AI + 教育赛道，帮助创作者快速完成：爆款对标 → 提示词生成 → AI 创作 → 一键发布 的全流程。

## 定位

- **差异化**：专注教育赛道，与码觉客公众号定位一致
- **核心价值**：让内容创作更高效

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue3 + Element Plus |
| 后端 | Spring Boot |
| 爬虫 | Python + Playwright |
| 数据库 | MySQL + Redis |
| 存储 | MinIO |
| 部署 | Docker Compose |
