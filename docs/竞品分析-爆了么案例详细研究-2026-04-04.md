## 三、完整工作流程（三步出爆款）

### 3.1 第一步：搜索热点选题

```
用户操作：
1. 在"爆款对标"页面查看当前什么赛道在爆
2. 发现感兴趣的文章 → "我也写这篇"
3. 输入标题（可以自己改写）

系统行为：
- AI 自动搜索全网相关素材
- 显示各来源的素材结果（白肚子、同行、新文等）
- 提供素材给用户参考和选择
```

**素材来源示例**：
- 白肚子（原文）
- 同行新文
- 各平台相关内容

### 3.2 第二步：设计人设提示词

```
用户操作：
1. 选择或创建人设（如"活跃的40岁宝爸"）
2. 投喂3-5篇同账号文章（方式二）
3. 系统分析后生成专属提示词

人设包含内容：
- 人物背景（年龄、职业、家庭情况）
- 写作风格（接地气/专业/幽默等）
- 叙事角度（个人经历/观点输出/故事讲述）
- 常用句式和表达方式
```

**人设示例**：
```
你是一个40岁的宝爸
有一个上小学的孩子
你在深圳打拼了10年
你写文章的角度是：从普通人逆袭的角度
你的风格是：接地气、有故事、真诚
```

### 3.3 第三步：AI生文 + 排版 + 发布

```
1. 点击"开始创作"
2. 等待不到1分钟
3. 生成完成，查看文章 + 配图
4. 点击"一键复制"
5. 打开公众号后台 → 粘贴 → 发布
```

**一键复制包含内容**：
- 标题（纯文本）
- 摘要（纯文本）
- 正文（HTML格式，可直接粘贴）
- 封面图（单独复制）

### 3.4 移动端操作流程（视频演示）

```
手机端打开小程序"教育谁到"
    ↓
找到要仿写的文章
    ↓
点击"擦个标题"（改写标题）
    ↓
选择配图模式（精准采集/原文采集）
    ↓
点击"开始创作"
    ↓
等待生成（不到1分钟）
    ↓
点击"查看"预览文章
    ↓
点击"复制"
    ↓
打开公众号APP → 写新文章 → 粘贴
    ↓
选择封面 → 预览 → 发布
```

## 四、提示词工程（核心技术壁垒）

### 4.1 提示词结构

```markdown
【人设定义】
角色：你是一个XX岁的XXX
背景：有XX经历，从事XX工作
特点：你的写作风格是XX，常用句式是XX

【素材】
（粘贴搜索到的素材内容）

【创作要求】
主题：（用户输入的标题）
字数：800-1500字
结构：开头XX、中间XX、结尾XX
风格：XX
情感：XX
```

### 4.2 人设设计要点

| 要素 | 说明 | 示例 |
|------|------|------|
| **基础信息** | 年龄、职业、家庭 | 40岁宝爸、深圳打拼10年 |
| **写作风格** | 接地气/专业/幽默 | 接地气、有故事、真诚 |
| **叙事角度** | 个人观点/经历分享 | 普通人逆袭的角度 |
| **情感基调** | 温暖/犀利/治愈 | 正能量、有共鸣 |
| **常用句式** | 口头禅/固定表达 | "说实话"、"其实我觉得" |

### 4.3 赛道分类与提示词策略

| 赛道类型 | 是否需要素材 | 提示词特点 |
|---------|-------------|-----------|
| 文学创作类（情感、星座、文案） | ❌ 无需素材 | 重人设、轻素材 |
| 实时信息类（娱乐、汽车、社会热点） | ✅ 需要素材 | 重素材、人设辅助 |
| 赚钱类、婚姻类 | ✅ 需要素材 | 重人设+素材组合 |

### 4.4 提示词优化技巧

1. **多投喂同账号文章**：投喂3-5篇效果优于单篇
2. **保持人设一致**：确定人设后不要频繁更换
3. **素材新鲜度**：热点话题需要最新素材
4. **风格提炼**：从样文中提炼出3-5个关键风格词

## 五、选题策略与爆款规律

### 5.1 发现高点击率文章的方法

| 方法 | 操作 | 效果 |
|------|------|------|
| **赛道监控** | 查看各赛道爆款列表 | 发现当前热门内容 |
| **竞品跟踪** | 关注对标账号的新文章 | 第一时间模仿 |
| **热点追踪** | 关注微博/抖音热点 | 蹭热点流量 |
| **评论区挖掘** | 看读者评论找共鸣点 | 发现真实需求 |

### 5.2 爆款标题规律

| 类型 | 示例 | 特点 |
|------|------|------|
| 数字型 | "三十岁不上班了" | 具体、有冲击力 |
| 疑问型 | "你还在XX吗？" | 引发好奇 |
| 对比型 | "别人XX，你却XX" | 制造反差 |
| 痛点型 | "XX的人，最后都XX了" | 直击痛点 |

### 5.3 关键发现：持续输出同一关键词

> **核心策略**：找到能爆的关键词，持续写！

```
示例：
选题"三十岁不上班了"爆了
    ↓
继续写"三十岁"相关话题
    ↓
写"三十岁跳槽"、"三十岁焦虑"等
    ↓
篇篇爆，今天报1万明天报10万
    ↓
系统推荐越来越精准
```

**原理**：微信推荐系统会学习用户的兴趣，持续输出同一关键词会让系统更精准地推荐给目标用户。

### 5.4 赛道选择经验

| 赛道 | 特点 | 成功率 |
|------|------|--------|
| 赚钱类 | 永恒话题、受众广 | 高 |
| 婚姻类 | 情感共鸣强 | 高 |
| 房子类 | 实用性强 | 中 |
| 体育类 | 波动大 | 低 |
| 育儿类 | 刚需强 | 高 |

## 八、对 zhi-mo 的具体建议

### 8.1 功能优先级（调整版）

根据爆了吗的功能，建议 zhi-mo 按以下优先级开发：

| 优先级 | 功能 | 原因 |
|--------|------|------|
| **P0** | AI生文 + 配图 | 核心价值，5分钟出文章 |
| **P0** | 一键复制 | 用户体验关键点 |
| **P0** | 人设提示词库 | 技术壁垒所在 |
| **P1** | 爆款对标列表 | 选题参考 |
| **P1** | 素材搜索 | 创作素材来源 |
| **P2** | 达人招募系统 | 后期变现 |

### 8.2 MVP 最小功能集

```
必须实现（才能跑通流程）：
1. 输入标题
2. 选择人设
3. 点击生成
4. 生成文章 + 配图
5. 一键复制
```

### 8.3 差异化方向

| 方面 | 爆了吗 | zhi-mo 差异化 |
|------|--------|---------------|
| 目标用户 | 全职达人 | 教育赛道创作者 |
| 赛道 | 多元化 | AI + 教育垂直 |
| 内容风格 | 追热点 | 深度、有价值 |
| 变现模式 | 达人招募 | 会员订阅 + cps |

### 8.4 技术难点

| 难点 | 解决方案 |
|------|---------|
| 微信文章抓取 | Playwright + 代理IP |
| AI生成质量 | 提示词工程优化 |
| 配图精准度 | 调用图库API |
| 内容合规 | AI预检 + 人工审核 |
