# 执墨 - AI 新媒体内容创作平台

## 一、项目概述

### 1.1 项目名称与定位

| 项目 | 内容 |
|------|------|
| **项目名称** | 执墨 |
| **slogan** | 执笔倚墨，AI成文 |
| **定位** | AI驱动的新媒体内容创作平台 |
| **目标赛道** | AI + 教育（与"码觉客"公众号定位一致） |
| **核心用户** | 自媒体创作者，尤其是教育类内容创作者 |

### 1.2 项目背景

- **差异化定位**：专注 AI + 教育赛道，做垂直领域的爆款创作工具
- **运营方**：码觉客公众号
- **后续维护**：飞飞（AI助手）+ 老大（开发者）

## 二、核心功能规划

### 2.1 四大核心功能

| # | 功能模块 | 核心价值 | 开发优先级 |
|---|---------|---------|-----------|
| 1 | 🔥 爆款对标 | 查询低粉爆款文章，为创作者提供参考 | P1 |
| 2 | ✍️ 提示词生成 | 学习对标账号风格，生成专属AI写作提示词 | P1 |
| 3 | 🤖 AI创作 | 根据提示词自动生成文章 + 配图 | P1 |
| 4 | 🚀 一键发布 | 推送文章到公众号草稿箱 | P2 |

### 2.2 功能详细说明

#### 功能1：爆款对标

**功能描述**：
- 每日更新低粉爆款文章（阅读量高但粉丝不高的文章）
- 按领域筛选：教育、AI、育儿、学习方法等
- 展示数据：标题、阅读量、点赞数、公众号名称、发布日期

**用户操作流程**：
```
浏览爆款列表 → 按领域筛选 → 收藏文章 → 生成提示词
```

**技术实现要点**：
- 爆款文章数据需要定时爬取（来源：搜狗微信 / 第三方平台）
- 数据存储：MySQL，文章表设计

---

#### 功能2：提示词生成

**功能描述**：
- 用户提交对标文章链接 → 系统分析文章风格 → 生成专属提示词
- 两种模式：
  - 单篇生成：适合快速体验
  - 多篇学习（推荐）：投喂3-5篇同账号文章，学习更充分

**核心逻辑**：
1. 爬取对标文章内容
2. 分析文章结构：开头方式、叙事手法、结尾套路
3. 提取标题规律（感叹号、疑问句、数字等套路）
4. 组装成结构化 Prompt
5. 用户输入新标题 → AI 按此风格生成内容

**赛道分类处理**：
| 赛道类型 | 是否需要素材 |
|---------|-------------|
| 文学创作类（情感、星座、文案） | 无需投喂素材 |
| 实时信息类（娱乐、汽车、社会热点） | 需要投喂素材 |

---

#### 功能3：AI创作

**功能描述**：
- 输入文章标题 + 提示词 → AI 生成完整文章
- 自动配图（两种模式：精准采集 / 原文采集）

**图片来源选择**：
| 模式 | 说明 |
|------|------|
| 精准采集 | AI 根据内容自动插入相关图片（调用图库API） |
| 原文采集 | 抓取原文章图片（已去除水印） |

**输出**：
- 完整文章正文
- 文章配图
- 在线排版预览

---

#### 功能4：一键发布

**功能描述**：
- 绑定公众号（管理员微信扫码授权）
- 将生成的文章推送到公众号草稿箱

**前提条件**：
- 需要认证的公众号（个人订阅号 API 权限有限）
- 管理员扫码授权

**用户操作**：
```
绑定公众号 → 选择目标公众号 → 确认推送 → 文章进入草稿箱
```

## 四、爆款文章数据获取方案

### 4.1 数据来源

| 来源 | 优点 | 缺点 |
|------|------|------|
| 搜狗微信搜索 | 免费、无需登录 | 数据老（2022年为主） |
| 微信开放平台API | 官方合规 | 需要认证公众号资质 |
| 第三方数据平台（新榜等） | 数据全 | 需要付费 |
| 用户主动提交链接 | 零成本 | 依赖用户主动输入 |

### 4.2 短期方案（推荐）

**用户提交微信文章链接 → 系统抓取内容**

```
用户粘贴微信文章链接
    ↓
Python 爬虫解析（Playwright 处理 JS 渲染）
    ↓
提取：标题、正文、作者、发布日期、配图
    ↓
存储到数据库
    ↓
作为 AI 创作的素材输入
```

### 4.3 长期方案

- 接入新榜 / 蝉妈妈 API 获取爆款数据
- 或定时爬取搜狗微信搜索结果（不保证实时性）

## 七、已识别的技术难点

### 7.1 高难度（需攻克）

| 难点 | 说明 | 可能的解决方案 |
|------|------|----------------|
| 微信文章抓取 | 微信有反爬机制 | Playwright 模拟浏览器 + 代理IP池 |
| 获取公众号全部历史文章 | 没有免费合规方案 | 用户主动提交链接，或接入第三方API |
| 一键发布到公众号 | 需要认证公众号资质 | 先用草稿箱功能，需要服务号 |
| 图片去水印 | 原图水印处理 | OpenCV 图像处理 or AI 消除 |

### 7.2 中等难度

| 难点 | 说明 | 可能的解决方案 |
|------|------|----------------|
| AI 内容合规检测 | 百度/阿里AI有内容审核 | Prompt 层做规避处理 |
| 提示词质量 | 如何生成高质量 Prompt | 人工调优 + 迭代 |

## 九（补充）、关键认知更新（2026-04-04）

### 9.1 微信公众号推荐机制变革

> **重要发现**：微信公众号 2023 年从订阅制改为推荐制

| 机制 | 说明 |
|------|------|
| **旧机制** | 粉丝才能看到文章，没有粉丝就没有阅读 |
| **新机制** | 系统根据用户兴趣推荐，AI 创作也能被推荐 |
| **影响** | 不需要粉丝积累，内容好就能爆，0 粉也能出爆款 |

### 9.2 爆了么核心功能（参考）

根据竞品分析视频，爆了么实现了以下核心功能：

| 功能 | 说明 |
|------|------|
| **爆款对标** | 查看当前什么赛道在爆、爆款文章列表 |
| **选题搜索** | 搜索热点/素材，支持多来源 |
| **素材库** | 收藏、整理素材 |
| **一键生文** | 输入标题 → AI 生成文章 + 自动配图（不到一分钟） |
| **精准采集** | AI 根据文章内容自动匹配配图 |
| **一键复制** | 复制标题、正文、封面到公众号后台 |
| **人设提示词** | 内置多种人设模板（活跃的爸爸/妈妈等） |

### 9.3 核心工作流（三步）

```
第一步：搜索热点选题
    ↓
    输入一个包含标题的选题（如"三十岁不上班了"）
    ↓
    AI 搜索全网素材，显示各来源结果

第二步：设计套人设提示词
    ↓
    选择人设（如"活跃的40岁宝爸"）
    ↓
    AI 提取该账号的写作风格、角度、背景
    ↓
    生成结构化的提示词

第三步：AI 生文 + 排版
    ↓
    点击"开始创作"
    ↓
    不到一分钟生成完整文章 + 配图
    ↓
    一键复制到公众号后台
```

### 9.4 关键发现：选题策略

> **持续输出同一关键词**：你写什么关键词，接下来写，他就能一直火

示例：选题"三十岁不上班了"爆了 → 继续写"三十岁"相关话题 → 篇篇爆
