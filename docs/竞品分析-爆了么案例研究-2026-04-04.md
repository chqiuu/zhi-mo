## 一、竞品案例：爆了么

### 1.1 案例背景

视频介绍了一对普通夫妻（深圳打工人，零基础）利用 AI 写公众号，**一年赚了 200 多万**：

| 指标 | 数据 |
|------|------|
| 第一篇文章数据 | 30多万阅读，1000多元收益 |
| 单日收益 | 最高一天 800+ |
| 粉丝数 | 几乎为零（系统推荐机制） |
| 团队规模 | 夫妻两人 |
| 达人招募 | 3000+ 人，5000+ 账号 |

**关键洞察**：微信公众号 **2023 年从订阅制改为推荐制**，AI 创作的文章可以被系统主动推荐给用户，**不需要粉丝也能爆**。

## 二、核心功能拆解（爆了么网站功能）

根据视频中展示的网站界面，爆了么实现了以下功能：

### 2.1 功能列表

| 功能 | 说明 | 对应 zhi-mo 模块 |
|------|------|------------------|
| 爆款对标 | 查看当前什么赛道在爆、爆款文章列表 | 🔥 爆款对标 |
| 选题搜索 | 搜索热点/素材，支持多来源 | ✍️ 提示词生成 |
| 素材库 | 收藏、整理素材 | 素材管理 |
| 一键生文 | 输入标题 → AI 生成文章 + 自动配图 | 🤖 AI创作 |
| 精准采集 | AI 根据文章内容自动匹配配图 | AI配图模块 |
| 一键复制 | 复制标题、正文、封面到公众号后台 | 🚀 一键发布 |
| 人设提示词 | 内置多种人设模板（活跃的爸爸/妈妈等） | ✍️ 提示词生成 |
| 赛道选择 | 赚钱类、婚姻类、房子类等多种赛道 | 赛道分类 |

### 2.2 爆了么核心流程（三步）

```
第一步：搜索热点选题
    ↓
    输入一个包含标题的选题（如"三十岁不上班了"）
    ↓
    AI 搜索全网素材
    ↓
    显示各来源的素材结果

第二步：设计套人设提示词
    ↓
    选择人设（如"活跃的40岁宝爸"）
    ↓
    AI 提取该账号的写作风格、角度、背景
    ↓
    生成结构化的提示词

第三步：AI 生文 + 排版
    ↓
    点击"开始创作"
    ↓
    不到一分钟生成完整文章 + 配图
    ↓
    一键复制到公众号后台
```

## 三、爆款文章发现机制

### 3.1 如何发现高点击率文章

| 方法 | 说明 |
|------|------|
| **赛道监控** | 查看当前什么赛道在爆（赚钱、婚姻、房子等） |
| **竞品文章列表** | 按赛道展示爆款文章列表 |
| **选题参考** | 看到好的文章 → "我也写这篇" |
| **标题规律** | 提取高点击率标题的套路（感叹号、疑问句、数字等） |

### 3.2 选题来源

视频中提到的选题来源：
- 用户自己发现的有趣话题
- 对标账号的爆款文章
- **系统推荐的赛道**（什么类型正在爆）

### 3.3 关键发现：持续输出同一关键词

> "你写什么关键词，接下来写，他就一直能火"

例如：选题"三十岁不上班了"爆了 → 继续写"三十岁"相关话题 → 篇篇爆

## 四、数据爬取方案

### 4.1 素材爬取

爆了么的素材搜索功能：
- **全网搜索**：输入选题后，AI 自动搜索全网相关内容
- **多来源展示**：显示各来源的素材结果
- **素材整合**：将搜索结果整理后提供给 AI 创作

### 4.2 爆款数据来源

| 来源 | 可行性 |
|------|--------|
| 搜狗微信搜索 | ✅ 免费，但数据较老 |
| 新榜 API | ⚠️ 需要付费 |
| 用户主动提交 | ✅ 零成本 |
| 定时爬取竞品平台 | ⚠️ 可能有法律风险 |

### 4.3 内容合规性

> "AI 创作的内容必须保证原创和真实，否则会被投诉"

解决方案：
- AI 生成后人工审核
- 使用 AI 内容检测工具预检
- 避免直接抄袭，加入个人观点和经历

## 六、提示词工程（核心壁垒）

### 6.1 人设提示词设计

爆了么内置的人设模板：

| 人设类型 | 特点 |
|----------|------|
| 活跃的宝爸/宝妈 | 接地气、有故事、真实感强 |
| 教授/专家 | 专业但枯燥 |
| 普通人逆袭 | 真实经历、情感共鸣 |

**关键点**：人设需要有**人物小传**（背景、年龄、职业、家庭情况）

### 6.2 提示词结构

```
【人设】
你是一个XX岁的XXX，有XX经历...
你写文章的角度是XX，背景是XX

【素材】
（粘贴搜索到的素材）

【要求】
根据素材，写一篇XX字的文章
风格：XX
结构：开头XX、中间XX、结尾XX
```

### 6.3 赛道分类处理

| 赛道类型 | 是否需要素材 |
|---------|-------------|
| 文学创作类（情感、星座、文案） | 无需素材 |
| 实时信息类（娱乐、汽车、社会热点） | 需要素材 |
| 赚钱类、婚姻类 | 需要素材 |

## 七、对 zhi-mo 项目的补充建议

### 7.1 功能优先级调整

根据爆了么的实际功能，建议调整 zhi-mo 的功能优先级：

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0 | **一键生文** | 核心功能：输入标题 → 生成文章+配图 |
| P0 | **精准配图** | AI 自动匹配文章内容的图片 |
| P0 | **一键复制** | 复制标题/正文/封面到公众号后台 |
| P1 | **人设提示词库** | 预设多种人设模板 |
| P1 | **素材搜索** | 搜索全网素材作为创作参考 |
| P2 | **爆款对标列表** | 按赛道展示爆款文章 |
| P2 | **达人招募系统** | 后期变现模式 |

### 7.2 核心差异化建议

| 方面 | 爆了么 | zhi-mo 可差异化的点 |
|------|--------|---------------------|
| 目标用户 | 全职达人 | 教育赛道垂直用户 |
| 赛道 | 多元化 | 专注 AI + 教育 |
| 内容风格 | 追热点 | 深度、有价值 |
| 变现模式 | 达人招募 | 会员订阅 + cps |

### 7.3 技术实现要点

根据视频展示的功能，需要重点实现：

1. **AI 生文速度**：不到一分钟生成完整文章
2. **配图精准度**：AI 根据内容自动匹配相关图片
3. **一键复制体验**：标题、正文、封面一键复制
4. **人设模板库**：预设多种人设，支持自定义

## 八、关键认知更新

### 8.1 微信公众号推荐机制

> **重要发现**：微信公众号 2023 年从订阅制改为推荐制

- **旧机制**：粉丝才能看到文章
- **新机制**：系统根据用户兴趣推荐，AI 创作也能被推荐
- **影响**：不需要粉丝积累，内容好就能爆

### 8.2 内容为王

- AI 只是工具，内容质量才是核心
- 需要持续测试不同赛道、选题
- **找到能持续爆的关键词**，然后围绕它写

### 8.3 最小 MVP

爆了么的 MVP 只需三步：
1. 输入标题
2. 点击生成
3. 复制发布
