## 一、需求概述

### 1.1 功能目标
实现公众号文章批量导入功能，支持：
1. **手动批量导入** — 用户提供多个文章链接，程序依次抓取
2. **剪贴板监控** — 后台自动检测文章链接并加入待处理队列

### 1.2 微信公众号文章抓取流程
```
用户复制文章链接 → 剪贴板监控识别 → 链接加入队列 → 解析文章正文 → 存储到素材库
```

### 1.3 技术选型
| 组件 | 技术方案 | 说明 |
|------|---------|------|
| 爬虫框架 | Spring WebClient + Jsoup | 轻量级，无需浏览器 |
| 浏览器模拟 | Selenium + Chrome Headless | 处理 JS 渲染（备用） |
| 剪贴板监控 | Java Clipboard API | 后台轮询监控 |
| 队列 | LinkedBlockingQueue | 内存队列，后续可扩展 Redis |
| 数据库 | MySQL + MinIO | 文章存储 + 图片资源 |

## 二、核心功能设计

### 2.1 文章抓取模块

#### 2.1.1 接口设计
```
POST /api/article/import
GET  /api/article/fetch?url={url}
POST /api/article/batch-import
```

#### 2.1.2 抓取流程
```
1. 接收文章 URL（微信文章链接格式: https://mp.weixin.qq.com/s/xxx）
2. 发起 HTTP 请求获取 HTML
3. Jsoup 解析正文内容：
   - 标题：<h1 id="activity-name">
   - 作者：<a id="js_name">
   - 正文：<div id="js_content">
   - 发布时间：<em id="publish_time">
   - 封面图：<img class="avatar">
4. 下载并上传图片到 MinIO
5. 保存文章到数据库
```

#### 2.1.3 微信公众号文章解析
```java
public class WechatArticleParser {
    
    public Article parse(String html) {
        Document doc = Jsoup.parse(html);
        
        // 提取标题
        String title = doc.select("#activity-name").first().text();
        
        // 提取作者（公众号名称）
        String author = doc.select("#js_name").first().text();
        
        // 提取正文（需要处理懒加载图片）
        Element content = doc.select("#js_content").first();
        // 替换懒加载图片 src
        content.select("img[data-src]").forEach(img -> {
            img.attr("src", img.attr("data-src"));
        });
        
        // 提取封面图
        String cover = doc.select(".avatar").first().attr("src");
        
        // 提取发布时间
        String publishTime = doc.select("#publish_time").first().text();
        
        return Article.builder()
            .title(title)
            .author(author)
            .content(content.html())
            .cover(cover)
            .publishTime(parseTime(publishTime))
            .build();
    }
}
```

### 2.2 剪贴板监控模块

#### 2.2.1 设计思路
后台运行一个定时任务，每秒检查剪贴板内容：
- 检测是否为微信文章链接（正则匹配 `https://mp.weixin.qq.com/s/`)
- 如果是新链接，加入待处理队列
- 记录已处理的链接（避免重复抓取）

#### 2.2.2 核心代码
```java
@Service
public class ClipboardMonitorService {

    private final BlockingQueue<String> articleQueue = new LinkedBlockingQueue<>();
    private final Set<String> processedUrls = ConcurrentHashMap.newKeySet();
    
    @PostConstruct
    public void startMonitoring() {
        ScheduledExecutorService executor = Executors.newSingleThreadScheduledExecutor();
        executor.scheduleAtFixedRate(this::checkClipboard, 0, 1, TimeUnit.SECONDS);
    }
    
    private void checkClipboard() {
        try {
            Clipboard clipboard = Toolkit.getDefaultToolkit().getSystemClipboard();
            Transferable transfer = clipboard.getContents(null);
            
            if (transfer.isDataFlavorSupported(DataFlavor.stringFlavor)) {
                String text = (String) transfer.getTransferData(DataFlavor.stringFlavor);
                
                // 检测微信文章链接
                if (isWechatArticleUrl(text) && !processedUrls.contains(text)) {
                    processedUrls.add(text);
                    articleQueue.add(text);
                    log.info("检测到新文章链接: {}", text);
                }
            }
        } catch (Exception e) {
            log.error("剪贴板监控异常", e);
        }
    }
    
    private boolean isWechatArticleUrl(String text) {
        return text != null && text.matches("https://mp\\.weixin\\.qq\\.com/s[\\?/].*");
    }
}
```

#### 2.2.3 队列消费
```java
@Scheduled(fixedDelay = 2000)
public void processQueue() {
    String url;
    while ((url = articleQueue.poll()) != null) {
        try {
            Article article = wechatArticleParser.fetchArticle(url);
            articleService.save(article);
            log.info("文章抓取成功: {}", article.getTitle());
        } catch (Exception e) {
            log.error("文章抓取失败: {}", url, e);
            // 失败重试机制
        }
    }
}
```
