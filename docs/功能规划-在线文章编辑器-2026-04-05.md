# 在线文章编辑器 - 功能规划

## 背景

目前生成的文章（如「AI时代，程序员到底是该焦虑还是该行动？」）是静态 HTML，需要手动复制到公众号后台。用户提出需要一个网页版在线编辑器，可以直接修改文章内容。

## 定位

执墨平台的核心工具之一，用于：
1. 编辑 AI 生成的文章草稿
2. 手动微调内容（满足"AI出初稿 + 人工微调"的工作模式）
3. 导出符合公众号格式的内容

## 核心功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| HTML 导入 | P0 | 粘贴已有文章 HTML 源码，解析出标题/摘要/正文 |
| 可视化编辑 | P0 | 分块编辑标题、摘要、正文 |
| 实时预览 | P0 | 编辑时实时预览公众号样式效果 |
| 一键复制 | P0 | 复制标题/摘要（纯文本）、正文（HTML） |
| 图片管理 | P1 | 上传/插入/删除文章中的图片 |
| 样式模板 | P2 | 切换不同的文章样式主题 |

## 技术选型

- **框架**：Vue3 + Element Plus（与项目一致）
- **富文本编辑器**：Tiptap（基于 ProseMirror，功能强大且可扩展）
- **状态管理**：Pinia
- **构建工具**：Vite

## 数据模型

```typescript
interface Article {
  id?: string;
  title: string;
  summary: string;
  content: string;        // HTML 格式的正文
  coverImage?: string;   // 封面图 URL
  author: string;
  createdAt: string;
  updatedAt: string;
}

interface EditorState {
  article: Article;
  isDirty: boolean;      // 是否有未保存的修改
  lastSaved?: string;    // 上次保存时间
}
```

## 导入/导出逻辑

### 导入 HTML 解析

输入的 HTML 通过 DOMParser 解析，提取：
- `<h1>` 或第一个 `<strong>` → 标题
- `<meta description>` 或特定 class → 摘要
- `<div class="content">` 或 `<article>` → 正文

### 导出格式

| 导出项 | 格式 | 用途 |
|--------|------|------|
| 标题 | 纯文本 | 公众号标题栏 |
| 摘要 | 纯文本 | 公众号摘要栏 |
| 正文 | HTML | 粘贴到公众号编辑器的「源码」模式 |
